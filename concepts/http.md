# HyperText Transfer Protocol (HTTP)

The standard protocol for requesting actions/resources on the web.

Anatomy: `VERB + URL + BODY`

Verbs: `POST`, `GET`, `PUT`, `PATCH`, `DELETE`

## Receive

Anatomy: `STAUS_CODE + BODY`

code | class
---- | ---
1xx  | informational
2xx  | success
3xx  | redirect
4xx  | client error
5xx  | server error
